\item Suppose that $f(x)$ is continuous and differentiable for all values of $x$, and suppose that $g(x) = e^{3 + f(x)^2}$.  Which of the following is $g'(x)$?
\vskip .5 cm
\begin{multicols}{2}
\begin{enumerate} \setlength{\itemsep}{.5cm}
\item $ g'(x) = \displaystyle e^{3+f(x)^2} 2f(x) f'(x)$ % correct
\item $ g'(x) = \displaystyle e^{3+f(x)^2}$
\item $ g'(x) = \displaystyle e^{3+f(x)^2} 2f(x)$
\item $ g'(x) = \displaystyle \frac{2f(x)f'(x)}{3 + f(x)^2}$ 
% \item None of the above
\end{enumerate}
\end{multicols}
