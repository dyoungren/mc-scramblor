\item For which value $c$ is the following function continuous on $(-\infty,\infty)$? \[
	f(x)=\begin{cases}
		\frac{1-\cos(2x)}{e^x-x-1} &\text{ if }x \ne 0\\
		c &\text{ if }x=0
	\end{cases}
\]
 \vskip .5 cm
\begin{multicols}{2}
    \begin{enumerate} \setlength{\itemsep}{.5cm}
          \item $c=4$ % correct
          \item $c=2$
          \item $c=1$
          \item $c=0$
		  % \item There is no value $c$ for which $f$ is continuous.
      \end{enumerate}
\end{multicols}%\newpage
