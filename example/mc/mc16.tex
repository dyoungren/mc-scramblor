\item Consider the curve defined implicitly by  \[
    x y^3+y^2=2 x+2.
\] 
Which expression below represents $\ds \frac{dy}{dx}$?
 \vskip .5 cm
\begin{multicols}{2}
    \begin{enumerate} \setlength{\itemsep}{.5cm}
          \item $\frac{2-y^3}{3 x y^2+2y}$ % correct
          \item $\frac{2}{y^3+3xy^2+2y }$
          \item $y^3-2$
          \item $\sqrt{\frac{2x+2}{x y+1}}$
		  % \item Cannot be determined with information given.
      \end{enumerate}
\end{multicols}%\newpage
